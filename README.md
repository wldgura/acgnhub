# ACGNHUB

ACGNHUB 是一个个人整理的 ACG（Anime / Comic / Game / Novel）导航网站。

该网站收集整理常用的二次元相关网站，包括：

- 动漫资源
- 漫画网站
- ACG社区
- 游戏资源
- 实用工具

目标是做一个简单、干净、好用的 ACG 导航站。
